# web

## Project setup
```
npm install
sudo npm install -g parcel-bundler
```

### Compiles and hot-reloads for development
```
parcel serve index.html --port 4200
```

### Compile with consistent filename for copy into site
```
parcel build --no-content-hash index.html
```
